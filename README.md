# Гонки на Pygame

**Описание**  
Простая 2D-игра, где вы управляете красной машиной, избегая столкновений с другими автомобилями. Дорога постоянно движется, а препятствия появляются случайным образом.

---

## Требования
- Python 3.6+
- Pygame (`pip install pygame`)

---

## Установка
1. Скачайте файл `roadcars.py` и запустите
